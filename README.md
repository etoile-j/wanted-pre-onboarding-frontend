## 프로젝트 실행 방법

배포 링크 혹은

```
npm install
```

```
npm start
```

## 기능

```
- 로그인/회원가입
유효성 검사
로컬 스토리지
로그인 여부에 따른 리다이렉트

- 투두 리스트
목록 불러오기
투두 추가, 수정, 삭제
체크유무로 완료 여부 표시
```
